# HRMS-PORTAL
# 🦸 MARVEL ✕ DC Kids Network

### HRMS + Entertainment Web Platform

## 📌 Overview

This project is a full-stack web application combining a kids entertainment platform with an internal Human Resource Management System (HRMS).

It includes a public-facing website and a secure internal dashboard for administrators and employees.

---

## 🚀 Features

### 🌐 Public Website

* Animated homepage
* Shows listing (card UI)
* About & Careers pages
* Contact form

### 🔐 Authentication

* Role-based login (Admin / Employee)

### 🏢 HRMS Dashboard

* Employee management
* Recruitment tracking
* Training module
* Announcements system

### 📊 Performance Analytics

* KPI metrics
* Score calculation system
* Charts & insights
* Leaderboard

---

## 🧮 Performance Score Formula

Final Score =
40% Task Score
20% Attendance
20% Training
20% Manager Rating

## ⚙️ Installation

### 1. Clone Repository

```bash
git clone https://github.com/your-username/marvel-dc-kids-network.git
cd marvel-dc-kids-network
```

### 2. Install Dependencies

```bash
cd client
npm install

cd ../server
npm install
```

### 3. Run Project

```bash
# Run backend
cd server
npm start

# Run frontend
cd client
npm start
```
